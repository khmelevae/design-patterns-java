# Реализация паттернов проектирования на Java
📚 Коллекция реализаций паттернов проектирования (GoF) на Java с примерами и пояснениями.
В репозитории есть наглядные примеры реализации классических паттернов проектирования, описанных в GoF.
Каждый паттерн имеет простой код.
Репозиторий сделал для изучения, повторения и использования в виде шпаргалки.
